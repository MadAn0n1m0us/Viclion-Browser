# Contributing

Thank you for contributing!

## Before submitting

- Follow PEP8.
- Test your code.
- Document public APIs.

## Pull Requests

- One feature per Pull Request.
- Keep commits clean.
- Explain why the change is useful.

## Code Style

Python 3.14+

QML 2.15+

PyQt5

## Commit Messages

feat:
fix:
docs:
